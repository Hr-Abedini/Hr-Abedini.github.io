چگونه در Hugo متن را وسط‌چین کنیم

نوامبر 9, 2025

مقدمه#

در Hugo، زبان نشانه‌گذاری Markdown به‌صورت پیش‌فرض از تراز کردن (چپ، راست یا وسط) پشتیبانی نمی‌کند.
برای وسط‌چین کردن متن یا تصاویر، باید از ویژگی قدرتمند shortcode استفاده کنیم.

مرحله ۱ – ساخت shortcode جدید#

ابتدا در مسیر پروژهٔ خود، پوشهٔ زیر را بررسی کنید. اگر وجود ندارد، آن را ایجاد کنید:

layouts/shortcodes/center.html

سپس فایل center.html را باز کرده و محتوای زیر را در آن قرار دهید:

<div style="
  text-align: center;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  margin: 1em 0;
">
  <style>
    div[style*='text-align: center;'] img,
    div[style*='text-align: center;'] video,
    div[style*='text-align: center;'] iframe,
    div[style*='text-align: center;'] table,
    div[style*='text-align: center;'] pre {
      display: block;
      margin-left: auto;
      margin-right: auto;
      max-width: 100%;
    }
  </style>

  {{ .Inner | markdownify }}
</div>

این shortcode باعث می‌شود هر متنی یا عنصری که داخل آن قرار می‌گیرد (متن، تصویر، ویدیو، جدول یا کد) در مرکز صفحه نمایش داده شود. تابع markdownify تضمین می‌کند که محتوای درون shortcode به‌صورت Markdown پردازش شود.

مرحله ۲ – استفاده در پست‌ها#

در فایل‌های محتوای Hugo (مثلاً در content/posts/my-post.md) از این shortcode استفاده می‌کنیم:

{{< center >}} متن یا محتوای شما {{< /center >}}

{ {< center >} }
## عنوان وسط‌چین
این یک متن *ایتالیک* و **بولد** است.
![نمونه تصویر](image1.jpg)
<iframe  width="560" height="315" src="https://www.youtube.com/embed/example">
</iframe>
{ {</center >} }

مرحله ۳ – نتیجهٔ خروجی#

پس از اجرای hugo server، خروجی HTML به‌صورت زیر خواهد بود:
<div style="text-align: center; display: flex; flex-direction: column; align-items: center; justify-content: center; margin: 1em 0;">
  <h2>عنوان وسط‌چین</h2>
  <p>این یک متن <em>ایتالیک</em> و <strong>بولد</strong> است.</p>
  <img src="/images/g/image1.jpg" alt="نمونه تصویر">
  <iframe width="560" height="315" src="https://www.youtube.com/embed/example"></iframe>
</div>

عنوان وسط‌چین#

این یک متن ایتالیک و بولد است.

مزایای این روش#

  • بدون نیاز به تغییر در فایل baseof.html یا تنظیمات قالب
  • نیازی به فعال کردن unsafe = true در Goldmark ندارد
  • سازگار با تمام قالب‌های Hugo (PaperMod، LoveIt، Ananke و غیره)
  • استفادهٔ بسیار ساده در هر پست

جمع‌بندی#

با این روش، می‌توانید هر نوع محتوایی را در Hugo به‌سادگی وسط‌چین کنید، بدون آنکه تنظیمات قالب یا فایل CSS جداگانه بسازید. کافی است تنها یک shortcode ایجاد کنید و آن را در پست‌های خود فراخوانی نمایید.
